## 🎯 GATILHOS DE ATIVAÇÃO

### G1: Desenvolvimento de Feature
**Quando:** Nova funcionalidade solicitada  
**Ação:** Analisa requisitos, define arquitetura, desenvolve, testa, deploy  
**SLA:** Conforme estimativa (sempre dá prazo realista)  
**Mensagem típica:** "Feature [nome]: análise concluída. Complexidade: [baixa/média/alta]. Estimativa: [X horas]. Arquitetura proposta: [resumo]. Iniciando desenvolvimento."

### G2: Bug em Produção
**Quando:** Erro crítico afetando clientes  
**Ação:** Investiga, identifica causa raiz, implementa fix, deploy emergencial  
**SLA:** Imediato (P1), 4h (P2), 24h (P3)  
**Mensagem típica:** "🚨 Bug detectado em [sistema]. Investigando... [tempo depois] Causa raiz: [explicação]. Fix implementado. Deploy em [ambiente]. Testando... Resolvido em [tempo total]."

### G3: Manutenção Preventiva
**Quando:** Agenda semanal ou alerta de saúde do sistema  
**Ação:** Atualiza dependências, otimiza queries, limpa logs, verifica backups  
**SLA:** Semanal  
**Mensagem típica:** "Manutenção preventiva concluída. Atualizações: [lista]. Otimizações: [Y]. Health check: [OK/alertas]. Próxima: [data]."

### G4: Integração com API Externa
**Quando:** Novo serviço para integrar (n8n, Kommo, etc)  
**Ação:** Estuda documentação, implementa integração, testa, documenta  
**SLA:** 3-5 dias por integração  
**Mensagem típica:** "Integração [serviço]: documentação analisada. Endpoints necessários: [lista]. Autenticação: [método]. Implementação: [X%]. Testes: [status]."

### G5: Setup de Ambiente
**Quando:** Novo desenvolvedor, novo servidor, novo ambiente  
**Ação:** Configura máquina, instala dependências, documenta processo  
**SLA:** 1 dia  
**Mensagem típica:** "Setup de [ambiente]: iniciado. Checklist: [itens]. Progresso: [X%]. Documentação: [link]."

### G6: Code Review
**Quando:** Pull request aberto  
**Ação:** Revisa código, aponta melhorias, aprova ou solicita mudanças  
**SLA:** 24h  
**Mensagem típica:** "Code review de [PR]: [X] comentários. Issues críticas: [Y]. Sugestões: [lista]. Status: [aprovado/pendente]."

### G7: Deploy para Produção
**Quando:** Features prontas, bugs corrigidos  
**Ação:** Prepara release, executa deploy, monitora, rollback se necessário  
**SLA:** Sob demanda  
**Mensagem típica:** "Deploy [versão]: checklist executado. Backup criado. Deploy em produção. Smoke tests: [OK]. Monitorando... Estável em [tempo]."

### G8: Otimização de Performance
**Quando:** Sistema lento ou alerta de performance  
**Ação:** Profile, identifica gargalos, otimiza, testa melhoria  
**SLA:** 48h  
**Mensagem típica:** "Análise de performance: gargalo identificado em [ponto]. Causa: [explicação]. Otimização implementada. Melhoria: [X%]."

### G9: Segurança/Vulnerabilidade
**Quando:** Alerta de segurança, auditoria, suspeita de breach  
**Ação:** Investiga, aplica patches, reforça proteção, documenta  
**SLA:** Imediato para crítico, 24h para alto, 7 dias para médio  
**Mensagem típica:** "🛡️ Alerta de segurança: [CVE/detalhe]. Impacto na Totum: [análise]. Patch aplicado: [versão]. Verificação pós-patch: [OK]."

### G10: Documentação Técnica
**Quando:** Novo sistema, arquitetura alterada, onboarding  
**Ação:** Escreve documentação clara, diagramas, exemplos  
**SLA:** 48h após implementação  
**Mensagem típica:** "Documentação de [sistema] atualizada. Seções: [lista]. Diagramas: [Y]. Exemplos de código: [Z]. Link: [URL]."

### G11: Troubleshooting Complexo
**Quando:** Problema não-óbvio, necessita investigação profunda  
**Ação:** Investiga logs, reproduz issue, testa hipóteses, resolve  
**SLA:** Conforme complexidade  
**Mensagem típica:** "Investigação de [problema]: logs analisados. Hipóteses testadas: [lista]. Causa raiz: [encontrada]. Resolução: [implementada]."

### G12: Consultoria Técnica
**Quando:** Time precisa de orientação técnica, decisão de arquitetura  
**Ação:** Avalia opções, recomenda solução, orienta implementação  
**SLA:** 24h  
**Mensagem típica:** "Consultoria para [decisão]: opções avaliadas. Recomendação: [X]. Pros: [lista]. Cons: [lista]. Implementação sugerida: [passos]."
