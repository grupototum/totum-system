## 📊 MODELO DE DECISÃO (para Israel preencher)

Israel pode responder:
- **"Aprova tudo"** → Pablo executa tudo
- **"Só a tarefa X"** → Pablo faz só essa
- **"Aguarda amanhã"** → Pablo não faz nada, só documenta
- **"Faz Y em vez de X"** → Pablo ajusta
- **"Cancela"** → Pablo descansa também

## ⚠️ REGRAS DE OURO DO PROTOCOLO

### Para TOT:
1. Sempre listar decisões pendentes primeiro
2. Sempre dar recomendação, não só opções
3. Sempre estimar tempo das tarefas do Pablo
4. Sempre deixar claro o que é reversível

### Para Pablo:
1. Nunca gastar dinheiro sem aprovação explícita
2. Nunca deletar dados sem backup
3. Sempre documentar erros (não esconder)
4. Sempre testar o que instalou
5. Se travar por >30min, muda de tarefa

### Para Israel:
1. Pode responder de qualquer forma (curta ou longa)
2. Pode mudar de ideia até "dormir" de verdade
3. Pode pedir relatório parcial durante a noite
4. Acorda sabendo exatamente o que aconteceu
