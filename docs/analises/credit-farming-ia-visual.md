## 🔍 ANÁLISE DETALHADA POR FERRAMENTA

### 1. Leonardo.AI ⭐⭐⭐⭐⭐
**URL:** https://app.leonardo.ai

**Pontos Fortes:**
- Modelos treinados específicos (Photon, Alchemy)
- Geração de assets para jogos/produtos
- Canvas infinito para edição
- Train your own models (LoRA)

**Créditos Free:**
- 150 tokens/dia (~30-50 imagens)
- Reset 00:00 UTC

**Plano Pro ($12/mês):**
- Tokens ilimitados (soft limit)
- Modelos premium
- API access
- Private generations

**Para Totum:**
- ✅ Ideal para criar assets visuais de clientes
- ✅ Melhor que Midjourney para controle criativo
- ✅ API permite integração automática

**Roteador:** Tarefas criativas complexas, concept art, variações de produto

### 2. Ideogram ⭐⭐⭐⭐
**URL:** https://ideogram.ai

**Pontos Fortes:**
- **Melhor do mercado para texto em imagem**
- Tipografia precisa (logos, posts com texto)
- Estilos variados (3D, pixel art, fotografia)

**Créditos Free:**
- 25 prompts/dia (~100 imagens em batch)
- 4 variações por prompt

**Plano Pro ($8/mês):**
- Prompts ilimitados
- Priority queue
- Private mode
- Edit/Remix

**Para Totum:**
- ✅ PERFEITO para posts sociais com texto
- ✅ Logos com tipografia
- ✅ Posts "frase do dia"

**Roteador:** Toda imagem que precisa de TEXTO legível
