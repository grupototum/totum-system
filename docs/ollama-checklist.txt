## Instalação

### 1. Instalar Ollama
```bash
curl -fsSL https://ollama.com/install.sh | sh
```
- [ ] Comando executado
- [ ] `ollama --version` retorna versão

### 2. Configurar Rede
```bash
sudo systemctl edit ollama.service
```
Adicionar:
```
[Service]
Environment="OLLAMA_HOST=0.0.0.0:11434"
```
- [ ] Arquivo criado
- [ ] `sudo systemctl restart ollama` executado
- [ ] `sudo systemctl status ollama` mostra "active"

### 3. Baixar Modelos
Execute um por vez:

```bash
ollama pull nomic-embed-text
```
- [ ] Baixado (274MB)

