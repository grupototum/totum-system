## 🔄 REGRA: CRIAÇÃO DE SKILLS

**Estabelecida por:** Israel  
**Data:** 2026-04-04  
**Aplicável a:** TODOS os agentes

### Regra
> "Toda tarefa que for bem executada e receber feedback positivo do usuário, o agente DEVE perguntar se deseja criar uma SKILL baseada naquela execução."

### Processo

```
1. Agente executa tarefa ✓
2. Usuário dá feedback positivo 👍
3. Agente pergunta: 
   "Posso salvar este processo como uma skill 
    para usar em tarefas similares no futuro?"
4. Se sim → Giles documenta na Alexandria
5. Skill fica disponível para todos os agentes
```

### Exemplo

**Cenário:** Pablo criou uma campanha de tráfego pago com excelente ROAS

```
Pablo: "A campanha gerou 5x ROAS. Quer que eu salve esta 
        estrutura de campanha como uma skill?"
        
Israel: "Sim"

Giles: [Registra skill no Banco de Skills]
        Nome: "Campanha ROAS 5x - Estrutura Padrão"
        Categoria: Tráfego Pago
        Tags: [meta-ads, alto-roas, b2b]
```
