### Funcionalidades Implementadas
- **Dual Mode**: Bot detecta automaticamente se usa Groq ou Ollama local
- **Classificador de demandas**: Funciona com ambos os backends
- **Geração de respostas**: Suporta system prompts em ambos
- **Fallback**: Se Groq falhar, retorna mensagem de erro amigável

### Vantagens do Groq vs Ollama Local
| Recurso | Ollama Local | Groq API |
|---------|--------------|----------|
| Velocidade | 2-5s | <1s ⚡ |
| RAM usada | ~2GB | 0MB |
| Setup | Complexo (Ollama + modelo) | Simples (API key) |
| Custo | Grátis | Grátis (tier generoso) |
| Privacidade | 100% offline | Nuvem (Groq) |
| Limites | Ilimitado | 20 req/min, 6k tokens/min |

### Tier Gratuito Groq
- 20 requisições/minuto
- 6.000 tokens/minuto
- 14.400 requisições/dia
- Modelo: Llama 3.2 (3B parâmetros)
