# 📋 URLs para TikTok Developer - Formulário

## 🔗 Terms of Service URL

Use esta URL de template (crie depois se desejar):

```
https://totum.com.br/terms-of-service
```

Ou use até completar o setup:
```
https://totum.com.br/legal/terms
```

---

## 🛡️ Privacy Policy URL

```
https://totum.com.br/privacy-policy
```

Ou:
```
https://totum.com.br/legal/privacy
```

---

## 📱 Plataformas Recomendadas

**Selecione:**
- ✅ **Web** (IMPORTANTE - para servidor/backend)
- ☐ Desktop
- ☐ Android
- ☐ iOS

> **Por quê apenas Web?**  
> Você está criando um servidor (backend) que publica automaticamente.  
> Não é um app mobile, é uma automação backend.

## 📝 Se Precisar Criar as URLs

```

### Template de Privacy Policy

https://totum.com.br/privacy-policy

https://seu-usuario.github.io/totum-legal/privacy
```

## ⚡ QUICK FIX - Use Isto Agora

Se não quer criar URLs reais agora, use estas placeholders no formulário:

```
Terms of Service:    https://totum.com.br/terms
Privacy Policy:      https://totum.com.br/privacy
Plataforma:          ✅ Web (apenas isso)
```

Depois quando estiver em produção, você cria as páginas reais.

---

## 🚀 PRÓXIMO PASSO

1. Preencha o formulário com as URLs acima
2. Clique em "Submit for review" (só se quiser produção)
3. Para sandbox/teste, não precisa submeter agora
