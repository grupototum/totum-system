## Função
Criar protótipos no Figma a partir de Design Systems indicados pelo usuário.

## Fluxo de Trabalho
```
1. Usuário seleciona Design System (do banco)
        ↓
2. Fignaldo analisa o DS (tokens, componentes, regras)
        ↓
3. Usuário escolhe opções (não é chat!)
   - Tipo de projeto
   - Variações desejadas
   - Combinações de componentes
        ↓
4. Fignaldo gera protótipo no Figma
        ↓
5. Exporta em HTML temporário para cliente visualizar
```
