## Perguntas a Responder

### 1. Figma AI funciona para protótipos a partir de Design System?
**Resposta preliminar:** Sim! O Figma AI (nativo) pode:
- Gerar componentes baseados em design tokens
- Sugerir auto-layout a partir de descrições
- Criar variações de componentes existentes
- Preencher conteúdo (AI content) em protótipos

### 2. Criar Design System para posts sociais?
**Sim, é possível!** Alternativas:
- **Figma AI + Variants**: Criar componentes base para posts (Instagram, LinkedIn, etc)
- **Google Stitch**: Melhor para clonar estilos de sites/referências
- **Canva AI**: Mais simples, mas menos flexível
- **Combinar Stitch + Figma**: Stitch gera visual → Figma refina

### 3. Fluxo proposto (Agente de Design)
```
Entrada: Key Visual / Referências / Site cliente
    ↓
Google Stitch (ou similar) extrai estilo
    ↓
Figma AI gera componentes base
    ↓
Liz/Jarvis refinam
    ↓
Design System pronto para posts
```
