## 🚀 FLUXO DE TRABALHO

1. Receber descrição do vídeo (prompt)
2. Estruturar cena por cena
3. Gerar código React/Remotion
4. Preview no navegador
5. Renderizar MP4
6. Entregar vídeo pronto

## 💡 USO

### Comando básico:
```bash
# Criar vídeo
chaplin criar "Stories 15s com logo Totum, fundo gradiente, texto animado"

# Renderizar
chaplin renderizar projeto/ stories_totum.mp4
```

### Prompts eficazes:
```
"Vídeo 1080x1920, 10 segundos:
- Fundo gradiente azul escuro → roxo
- Logo no topo (fade in 1s)
- Texto 'Oferta Especial' com bounce
- Contador regressivo 5s no final"
```
